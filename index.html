Tópico 1: Variável (Mitigação de Churn)

Estratégia para converter clientes com potencial de churn para o modelo variável.

OKRs (Resultados-Chave)

KR1
Converter 50% dos clientes com potencial de churn para o modelo variável até o fim do Q1
KR2
Fechar 5 novos clientes em nichos (E-commerce/Delivery) já no modelo variável até o fim do Q1
KR3
Atingir 80% de satisfação em modelo variável até o final de Janeiro

Atividades Principais

  • Mapear clientes com potencial churn (critérios: queda de resultado, reclamações)
  • Estruturar oferta variável (definir pitch de vendas)
  • Iniciar prospecção em nichos priorizados

Próximos Passos

  • Mapear clientes com potencial churn
  • Gerar resultado (base para o variável)
  • Oferecer serviço variável para clientes com churn
